## Введение

Система автоматизации управления контентом предназначена для автоматического сбора новостей, их рерайта и публикации в социальных сетях. Система поддерживает работу с Twitter, LinkedIn, YouTube и Threads, позволяя управлять контентом для 13 аккаунтов с учетом индивидуального tone of voice для каждого из них.

Данное руководство поможет вам начать работу с системой и эффективно использовать все её возможности.

## Начало работы

### Вход в систему

1. Откройте веб-браузер и перейдите по адресу вашей системы (например, http://social-automation.example.com)
2. Введите имя пользователя и пароль (по умолчанию: admin/admin)
3. После успешной авторизации вы будете перенаправлены на главную страницу системы

### Первоначальная настройка

При первом запуске системы рекомендуется выполнить следующие действия:

1. Изменить пароль администратора (меню "Настройки" -> "Профиль")
2. Проверить настройки подключения к API социальных сетей (меню "Настройки" -> "API")
3. Настроить источники новостей (меню "Настройки" -> "Источники")
4. Настроить шаблоны tone of voice для каждого аккаунта (меню "Настройки" -> "Tone of Voice")

## Управление источниками новостей

### Просмотр источников

Для просмотра настроенных источников новостей:

1. Перейдите в меню "Настройки" -> "Источники"
2. На странице отобразится список всех настроенных источников с их статусом

### Добавление нового источника

1. На странице источников нажмите кнопку "Добавить источник"
2. Заполните форму:
   - **Название** - название источника для отображения в системе
   - **URL** - адрес источника новостей
   - **Тип парсера** - выберите тип парсера из списка
   - **Ключевые слова** - укажите ключевые слова для фильтрации новостей (опционально)
   - **Частота обновления** - как часто проверять источник на новые новости
3. Нажмите "Сохранить"

### Редактирование источника

1. На странице источников найдите нужный источник и нажмите кнопку "Редактировать"
2. Внесите необходимые изменения
3. Нажмите "Сохранить"

### Удаление источника

1. На странице источников найдите нужный источник и нажмите кнопку "Удалить"
2. Подтвердите удаление

## Настройка аккаунтов социальных сетей

### Просмотр аккаунтов

Для просмотра настроенных аккаунтов:

1. Перейдите в меню "Аккаунты"
2. На странице отобразится список всех настроенных аккаунтов с их статусом

### Добавление нового аккаунта

1. На странице аккаунтов нажмите кнопку "Добавить аккаунт"
2. Заполните форму:
   - **Название** - название аккаунта для отображения в системе
   - **Социальная сеть** - выберите социальную сеть из списка
   - **API ключи** - введите необходимые API ключи (зависит от выбранной социальной сети)
   - **Шаблон tone of voice** - выберите шаблон tone of voice из списка
3. Нажмите "Сохранить"

### Проверка статуса аккаунта

1. На странице аккаунтов найдите нужный аккаунт
2. Нажмите кнопку "Проверить статус"
3. Система выполнит проверку подключения к API и отобразит результат

### Отключение аккаунта

Если вы хотите временно отключить публикацию в определенный аккаунт:

1. На странице аккаунтов найдите нужный аккаунт
2. Переключите статус в положение "Отключен"
3. Нажмите "Сохранить"

## Настройка tone of voice

### Просмотр шаблонов

Для просмотра настроенных шаблонов tone of voice:

1. Перейдите в меню "Настройки" -> "Tone of Voice"
2. На странице отобразится список всех настроенных шаблонов

### Добавление нового шаблона

1. На странице шаблонов нажмите кнопку "Добавить шаблон"
2. Заполните форму:
   - **Название** - название шаблона для отображения в системе
   - **Стиль** - выберите стиль из списка (casual, professional, informative, etc.)
   - **Промпт для OpenAI** - введите промпт для системы рерайта
   - **Примеры** - добавьте примеры текстов в нужном стиле (опционально)
3. Нажмите "Сохранить"

### Тестирование шаблона

1. На странице шаблонов найдите нужный шаблон и нажмите кнопку "Тестировать"
2. Введите тестовый текст
3. Нажмите "Запустить тест"
4. Система выполнит рерайт текста с использованием выбранного шаблона и отобразит результат

## Запуск автоматизации

### Ручной запуск

1. Перейдите в меню "Автоматизация" -> "Запустить"
2. Настройте параметры запуска:
   - **Источники** - выберите источники новостей из списка
   - **Аккаунты** - выберите аккаунты для публикации
   - **Ключевые слова** - укажите ключевые слова для фильтрации новостей (опционально)
   - **Максимальное количество новостей** - ограничьте количество обрабатываемых новостей
   - **Задержка между публикациями** - укажите задержку в минутах
3. Нажмите "Запустить"
4. Система начнет процесс автоматизации и отобразит прогресс

### Настройка расписания

Для настройки автоматического запуска по расписанию:

1. Перейдите в меню "Автоматизация" -> "Расписание"
2. Нажмите кнопку "Добавить задачу"
3. Настройте параметры задачи:
   - **Название** - название задачи для отображения в системе
   - **Расписание** - выберите частоту запуска (ежедневно, еженедельно, ежемесячно)
   - **Время запуска** - укажите время запуска
   - **Параметры запуска** - настройте параметры аналогично ручному запуску
4. Нажмите "Сохранить"

## Устранение неполадок

### Проблемы с парсером

Если система не собирает новости или собирает некорректные данные:

1. Проверьте доступность источников новостей
2. Перейдите в меню "Логи" и проверьте логи парсера
3. Убедитесь, что структура HTML на сайте-источнике не изменилась
4. Попробуйте обновить парсер или создать новый

### Проблемы с рерайтом

Если система не выполняет рерайт или результаты рерайта некачественные:

1. Проверьте настройки интеграции с Make.com
2. Перейдите в меню "Логи" и проверьте логи рерайта
3. Убедитесь, что API ключи для OpenAI указаны корректно
4. Попробуйте изменить промпт для рерайта

### Проблемы с публикацией

Если система не публикует контент в социальные сети:

1. Проверьте статус аккаунтов через меню "Аккаунты"
2. Перейдите в меню "Логи" и проверьте логи публикации
3. Убедитесь, что API ключи для социальных сетей указаны корректно и не истек срок их действия
4. Проверьте, не достигнуты ли лимиты API
5. Попробуйте увеличить задержку между публикациями
